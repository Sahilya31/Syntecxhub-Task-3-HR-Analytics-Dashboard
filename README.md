# HR Analytics Dashboard

This repository contains an interactive **HR Analytics Dashboard** built as part of the Syntecxhub Internship (Task 3). The project demonstrates how to handle employee data, store it in a relational database, and visualize key HR metrics dynamically.

## 🚀 Features
* **Dynamic Department Filter:** A dropdown slicer to switch between 'All Departments' or specific teams (Engineering, Sales, HR, Marketing, Finance).
* **Live KPI Metrics:** Real-time calculation of Total Headcount, Attrition Rate (%), and Average Annual Salary.
* **Retention Analysis:** A clean pie chart showing the proportion of Active vs. Resigned employees.
* **Attrition Breakdown:** A bar chart displaying attrition rates across different departments or specific job roles.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Database:** SQLite (`sqlite3` in-memory database)
* **Data Manipulation:** Pandas
* **Visualization:** Matplotlib, Seaborn
* **Interactivity:** IPywidgets (`widgets`)

## 📊 How to Run the Dashboard
1. Open the `Untitled81.ipynb` notebook in Google Colab.
2. Run all the code cells (`Ctrl + F9`).
3. Use the **Dept Slicer** dropdown at the bottom to interact with the dashboard.
